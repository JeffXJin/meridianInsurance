# React Node Starter

A simple starter project using React, Express, MongoDB and AntDesign.

![](https://user-images.githubusercontent.com/5960988/46710580-0ca53f00-cc7b-11e8-8328-f49e0a14c601.png)

## Installing

```
$ npm install
```

## Running The App

### Dev

on port 3000.

```
$ npm run start
```

### Production

on port 3001.

```
$ npm run build
$ node server/server.js
```
